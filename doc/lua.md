lua
========
[LUA 5.3.2 ](http://www.lua.org/ftp/lua-5.3.2.tar.gz)

## Install
----------
```
yum install gcc readline readline-devel
tar zxvf lua-5.3.2.tar.gz
cd lua-5.3.2 && make linux

debian:
apt-get install lua5.3
```

## run
----------

lua5.3
